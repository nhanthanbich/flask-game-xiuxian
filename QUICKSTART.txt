═══════════════════════════════════════════════════════════════
   WEB VERSION (Flask) - KHUYÊN DÙNG ⭐
═══════════════════════════════════════════════════════════════

1️⃣ CÀI ĐẶT:
    pip install Flask

2️⃣ CHẠY:
    python3 flask_app.py

    HOẶC tự động:
    ./run_web.sh      (Linux/macOS)
    run_web.bat       (Windows)

3️⃣ TRUY CẬP:
    Mở trình duyệt: http://localhost:5000

═══════════════════════════════════════════════════════════════
   CẤU TRÚC FILES
═══════════════════════════════════════════════════════════════

flask_app.py          ← Chạy web version
main.py               ← Chạy terminal version

templates/            ← HTML templates (UI đã nâng cấp)
  start.html          ← Màn hình bắt đầu
  new_game.html       ← Chọn tộc hệ

engine/               ← Backend logic (KHÔNG SỬA)
ui/                   ← Terminal UI (KHÔNG SỬA)
data/                 ← Game data (KHÔNG SỬA)

   TROUBLESHOOTING
